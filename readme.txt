== Description ==

Cart2Chat helps WooCommerce stores manage personalized product sales through WhatsApp.

It includes:

* Product types and custom fields per type
* Conditional fields ("show only if")
* Product-page WhatsApp order flow
* Checkout WhatsApp payment method flow
* WhatsApp customer/shipping form builder
* Visual design settings for forms and buttons
* English-first UI with Spanish translations

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/` or install the ZIP from `Plugins > Add New > Upload Plugin`.
2. Activate the plugin through the `Plugins` menu.
3. Make sure WooCommerce is active.
4. Go to `Cart2Chat` in the admin sidebar and configure:
5. `General`: usage mode, WhatsApp number, opening/closing messages, WhatsApp form fields.
6. `Catalog & Fields`: product types and custom fields.
7. `Design`: form theme and button color settings.

== Frequently Asked Questions ==

= Can I use Cart2Chat without checkout? =

Yes. Use `Product page only` mode to send orders directly via WhatsApp from product pages.

= Can I use Cart2Chat with checkout? =

Yes. Use `Checkout only` or `Both` and enable the Cart2Chat WhatsApp payment method in WooCommerce payments.

= Can I create my own product types and fields? =

Yes. In `Catalog & Fields` you can create product types, add fields, reorder them, and configure conditional visibility.

= Does Cart2Chat support Spanish? =

Yes. The plugin is native in English and includes Spanish (`es_ES`) translations.

= Can the catalog be empty? =

Yes. You can leave product types empty and configure them later.
